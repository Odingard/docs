# What is VERDICT WEIGHT?

**VERDICT WEIGHT** is a context-adaptive, multi-source confidence synthesis framework for autonomous AI intelligence systems. It defines the mathematical methodology for transforming raw intelligence signals from multiple independent sources into calibrated, adversarially-robust confidence scores.

## Formal Definition

VERDICT WEIGHT is a scoring function **f: (SR, CC, TD, HA, O) -> (SS, DI, CW)** where:

- **SR** — Source Reliability score in [0,1]
- **CC** — Corroboration Count score in [0,1]
- **TD** — Temporal Decay score in [0,1]
- **HA** — Historical Accuracy score in [0,1]
- **O** — Context profile (domain-specific weight vector)
- **SS** — Signal Strength output in [0,1]
- **DI** — Doubt Index output in [0,1]
- **CW** — Consequence Weight output in [0,1]

The framework specifies:
1. How each input stream is computed from raw data
2. How streams are synthesized into SS via context-weighted geometric mean
3. How DI is computed from inter-stream disagreement
4. How CW integrates SS and DI into an actionable score
5. How context profiles adapt weights to operational domains

## The Three-Layer Architecture

VERDICT WEIGHT is designed to operate as a confidence layer that sits between raw intelligence ingestion and the decision engine:

```
Raw Intelligence Sources
         |
  [VERDICT WEIGHT]
  |-- Source Reliability scoring
  |-- Corroboration synthesis
  |-- Temporal decay calculation
  |-- Historical accuracy integration
  |-- Context-adaptive weighting
         |
  SS / DI / CW outputs
         |
  Decision Engine / Agent / Policy Layer
```

## The Core Insight

The key mathematical insight behind VERDICT WEIGHT is that **corroboration is the dominant predictive stream** for adversarial signal detection. Across 10,000 validation scenarios, corroboration count was the single strongest predictor of ground truth accuracy — more than source credibility, more than recency.

This means: an adversary can manufacture a credible source. They cannot manufacture independent corroboration at scale without triggering the Doubt Index.

That asymmetry is the moat.
