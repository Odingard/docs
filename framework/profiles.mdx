# Context Profiles

Different operational domains have fundamentally different signal reliability dynamics. A weight profile optimized for cybersecurity threat intelligence produces suboptimal results in healthcare diagnostics. VERDICT WEIGHT™ solves this with 12 validated context profiles.

## Profile Architecture

Each profile specifies four weights (summing to 1.0) and a domain-specific temporal decay rate (λ):

```python
@dataclass
class WeightProfile:
    W_SR: float      # Source Reliability weight
    W_CC: float      # Corroboration Count weight
    W_TD: float      # Temporal Decay weight
    W_HA: float      # Historical Accuracy weight
    lambda_td: float # Temporal decay rate
    doubt_penalty: float = 0.3
```

Best for: SOC operations, threat feeds, vulnerability intelligence

## Custom Profile Creation

Organizations can define custom profiles for proprietary domains:

```python
from verdict_weight import WeightProfile, VerdictWeight

custom_profile = WeightProfile(
    W_SR=0.40,
    W_CC=0.30,
    W_TD=0.15,
    W_HA=0.15,
    lambda_td=0.12,
    doubt_penalty=0.35
)

vw = VerdictWeight(profile=custom_profile)
```
