# Research Directions

VERDICT WEIGHT™ v3.0 addresses multi-source confidence synthesis. Several extensions are in active development.

## Causal VERDICT WEIGHT™

**Status: In development**

The base framework answers: *"How confident should we be in this signal?"*

Causal VERDICT WEIGHT™ answers: *"Which evidence stream caused this confidence level — and what would change it?"*

Using Pearl's do-calculus and Shapley value attribution, Causal VERDICT WEIGHT™ decomposes each CW score into stream-level causal contributions. This enables:

- Explainable confidence decisions for regulated environments
- Adversarial manipulation detection at the stream level
- Targeted corroboration requests ("we need more CC to elevate this signal")

The formal Structural Causal Model (SCM) is defined. The Shapley attribution layer is in implementation.

---

## Temporal VERDICT WEIGHT™

**Status: Research phase**

The base framework scores a single point in time. Temporal VERDICT WEIGHT™ scores the *evolution* of confidence over time.

An intelligence signal that starts low and builds corroboration over 72 hours is categorically different from one that starts high and decays. Time-series confidence scoring — tracking how SS, DI, and CW move — is a primitive that does not yet exist.

This represents a second independent paper, trademark, and patent.
