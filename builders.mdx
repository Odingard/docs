## Integration Patterns

### Pattern 1: Pre-Decision Scoring

Insert VERDICT WEIGHT™ between your intelligence ingestion layer and your decision engine:

```
Intelligence Sources → [VERDICT WEIGHT™] → Decision Engine
```

Only signals above your CW threshold reach the decision layer. Signals with high DI are flagged for review.

### Pattern 2: RAG Confidence Gating

Score retrieved documents before injecting into LLM context:

```python
retrieved_docs = retriever.get(query)
for doc in retrieved_docs:
    result = vw.score(
        SR=doc.source_reliability,
        CC=doc.corroboration_count,
        TD=doc.temporal_score,
        HA=doc.historical_accuracy
    )
    if result.consequence_weight >= 0.45:
        context.append(doc)
```

### Pattern 3: Alert Triage

Use CW tiers to prioritize analyst workload:

```python
result = vw.score(SR, CC, TD, HA)

if result.tier == "CRITICAL":
    alert_manager.escalate_immediately(signal)
elif result.tier == "HIGH":
    alert_manager.queue_priority(signal)
elif result.tier in ["MEDIUM", "LOW"]:
    alert_manager.queue_standard(signal)
else:
    alert_manager.archive(signal)
```
