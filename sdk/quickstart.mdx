# Quickstart

## Installation

```bash
pip install verdict-weight
```

## Basic Usage

```python
from verdict_weight import VerdictWeight, ContextProfile

# Initialize with a domain profile
vw = VerdictWeight(profile=ContextProfile.CYBERSECURITY)

# Score an intelligence signal
result = vw.score(
    SR=0.85,  # Source Reliability
    CC=0.72,  # Corroboration Count
    TD=0.90,  # Temporal Decay
    HA=0.78   # Historical Accuracy
)

print(result.consequence_weight)  # 0.801
print(result.tier)                # "CRITICAL"
print(result.signal_strength)     # 0.823
print(result.doubt_index)         # 0.089
```

## Detecting Adversarial Injection

```python
# Same credible source — but no corroboration
result = vw.score(SR=0.92, CC=0.08, TD=0.88, HA=0.85)

print(result.tier)          # "LOW"
print(result.doubt_index)   # 0.731  ← HIGH
print(result.warning)
# "WARNING: High inter-stream disagreement (DI=0.731).
#  High source credibility with near-zero corroboration
#  is the primary signature of adversarial injection."
```
